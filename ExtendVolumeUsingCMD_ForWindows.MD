#### Get disk details for windows using cmd:
````sh
get-disk
````
#### Volume Details:
````sh
get-volume
````
#### Partition Details:
````sh
get-partition
````
#### Add volume using below commands on cmd:
````sh
 diskpart
 list volume 
 select volume 1
 extend size 500000
 ````
 Note: Please select the volume as per the request ex: select volume 1 and extend volume to 5GB.
